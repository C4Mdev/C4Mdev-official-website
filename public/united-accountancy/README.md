# United Accountancy — images

Put your real assets in **this folder** (`public/united-accountancy/`).

| File | Use |
|------|-----|
| `hero.jpg` or `hero.png` | Main project image (works listing + project detail hero) |
| `gallery-1.jpg` … `gallery-3.jpg` | The three gallery thumbnails on the project page |

After adding files, update paths in **`src/data/portfolioProjects.ts`** for the `united-accountancy` entry (`image` and `galleryImages`).

Until then, the site uses **`placeholder.svg`**.
