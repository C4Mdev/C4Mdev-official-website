# Other Project — images

Use this folder for projects that don’t have a dedicated folder yet.

| File | Use |
|------|-----|
| `hero.jpg` or `hero.png` | Main project image |
| `gallery-1.jpg` … `gallery-3.jpg` | Gallery thumbnails |

Update paths in **`src/data/portfolioProjects.ts`** when you add a new project.

Until then, the site uses **`placeholder.svg`**.
